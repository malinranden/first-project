How to start a new project:

  1. Create a file in computer
  2. Add two folders images and styles
  3. Open file/mapp through visual studio code
  4. Create an index.html file inside folder + a new folder called styles + a file in styles called index.css (So you should have a folder with: images folder, styles folder and pages folder. And an index.html file. And inside styles folder a file called index.css)
  5. Create repository on Github
  6. Add to terminal - copy paste the commands on GitHub (thats how the back end teacher did when he helped me… in the info and videos Michelle does it in steps - either way ok?)
  7. Create basic HTML markup: write ”html” and choose html:5 - start creating content
